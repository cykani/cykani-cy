## Launch

```typescript
import { strike } from "cykani-stealth";

const session = await strike({ humor: true });
```

Binary auto-downloads on first launch.

## Navigate

```typescript
await session.goto("https://example.com");

const title = await session.page().title();
console.log(title);
```

## AutoPilot (No Selectors)

```typescript
await session.goto("https://example.com/login");

const ap = session.autoPilot();
await ap.signIn({
  email: "user@example.com",
  password: "secret",
});
```

## Close

```typescript
await session.close();
```

## Full Example

```typescript
import { strike } from "cykani-stealth";

async function main() {
  const session = await strike({ humor: true });
  await session.goto("https://example.com");

  const title = await session.page().title();
  console.log("Page title:", title);

  await session.close();
}

main();
```
